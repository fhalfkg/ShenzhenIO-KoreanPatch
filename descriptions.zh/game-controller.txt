* *x*と*y*はジョイスティックに接続された単純入力．
* *a*と*b*はボタンに接続された単純入力．
* *radio-rx *は無線受信機に接続されたノンブロッキングXBus入力．
* *radio-tx*は無線送信機に接続されたXBus出力．
* ゲーム機から無線でデータパケットを受信したら，必ず最新の入力状態を3値のデータパケットで送信する．

#############

|     |     | 0, if *a* = 0 and *b* = 0
| *x* | *y* | 1, if *a* = 100 and *b* = 0
|     |     | 2, if *a* = 0 and *b* = 100
|     |     | 3, if *a* = 100 and *b* = 100
